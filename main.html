Time and Space Complexity

Time complexity is a way to analyze how efficient
an algorithm is in terms of the time it takes to run
as a function of the size of the input. It provides
a high-level understanding of how the runtime of
an algorithm grows as the input size increases.

Space Complexity is a measure of the amount of working
storage an algorithm needs. It quantifies the total space
required by the algorithm as a function of the size of the input data.

O(n): Linear space - the space required grows linearly with the input size.
